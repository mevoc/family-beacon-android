# Family Beacon

**Family Beacon** is an open-source Android app designed for families to share safety signals using SMS and local device features.

The app allows trusted contacts (for example parents) to request status information such as location or battery level, even when mobile data is unavailable.

## Key principles

- ✅ Explicit opt-in for every feature
- ✅ No cloud services, no servers
- ✅ SMS and local device features only
- ✅ Transparent activity log visible in the app
- ✅ Designed for family safety, not surveillance

## Features

- 📍 SMS-based location request (from whitelisted numbers)
- 🔋 Battery level alerts
- 🚨 Emergency alert (audible + vibration)
- 🗺️ Geofencing (safe zones)
- 📜 Local event log (visible on the device)

All features can be enabled or disabled individually.

## Ethics & consent

Family Beacon is **not** designed for covert monitoring.

- The device user should be aware of what the app does
- All sensitive actions require explicit consent
- Configuration changes are protected by the device lock
- The app can be disabled or uninstalled at any time

See [ETHICS.md](ETHICS.md) and [PRIVACY.md](PRIVACY.md) for details.

## Installation

Currently distributed as a downloadable APK (outside Google Play).

1. Go to [Releases](https://github.com/mevoc/family-beacon-android/releases/latest)
2. Download `app-release.apk`
2. Enable installation from unknown sources (Android setting)
4. Install and follow the in-app consent flow
