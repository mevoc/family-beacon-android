# Privacy Policy – Family Beacon

Family Beacon is designed with privacy as a core principle.

## Data collection

Family Beacon does **not** collect or store personal data on external servers.

The app may access the following data **only when explicitly enabled**:

- Device location (for SMS reply or geofence events)
- Battery level
- Incoming SMS (limited to whitelisted numbers)

## Data storage

- All settings are stored locally on the device
- Event logs are stored locally using an on-device database
- No data is uploaded to the internet or third parties

## Data sharing

- Data is shared **only** via SMS
- SMS messages are sent **only** to user-approved (whitelisted) numbers
- No analytics, ads, or trackers are used

## User control

- All features are opt-in
- Features can be disabled at any time
- The app can be uninstalled at any time

## Children

Family Beacon is intended for family use.
Parents or guardians are responsible for ensuring appropriate and transparent use on a child’s device.

## Contact

If you have questions about privacy, please open an issue in the project repository.
